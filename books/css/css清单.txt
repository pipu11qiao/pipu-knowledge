##### 问题：const key in obj 和 const key of obj 的区别

**key in**用于对象，会遍历继承的属性，需要hasOwnProperty来守护,不保证遍历键的顺序 ,**key of**用于数组和可迭代对象不会遍历继承的属性,可以保证遍历键的顺序.

## 学习与实践建议
1. **浏览器开发者工具** 是最快的反馈回路，多用 “Inspect Element” 拖改属性。  
2. **做一个 Tiny-UI 库**：按钮、输入框、卡片，从样式到交互全链路练手。  
3. **CodePen/PlayCode 日更小练习**：每天用一种 CSS 特性实现一个小效果。  
4. **阅读优秀源码**：Tailwind、Bootstrap、Material-UI，都把 CSS 工程化思路写给你看。  

把上面 12 大块逐一过一遍，配合动手实验，你就能把常用 CSS 基础打牢，面对设计稿或重构旧站都能游刃有余。祝学习顺利！
